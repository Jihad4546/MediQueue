# MediQueue - Your Personal Learning Companion

**MediQueue** is a modern, smart online learning platform designed to bridge the gap between students and expert educators. Our platform enables students to find top-tier tutors and book one-on-one live sessions seamlessly using our intuitive digital booking system.

### 🚀 Live Site URL:
[https://medi-queue-six.vercel.app]

---

### ✨ Key Features:

* **One-on-One Live Sessions:** Connect directly with expert tutors via high-quality video calls for personalized learning and doubt resolution.
* **Instant Digital Token Booking:** Eliminate manual scheduling conflicts with our automated token system, ensuring real-time slot availability and instant confirmation.
* **Verified Mentor Network:** Access a diverse pool of 100+ verified tutors and industry experts across various academic and professional disciplines.
* **Seamless User Dashboard:** Track your upcoming sessions, manage bookings, and monitor your personal learning progress through an intuitive dashboard.
* **All-in-One Learning Ecosystem:** From core academic subjects to modern skill development, everything you need is available under one roof.

---

### 🛠 Tech Stack:
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS, HeroUI
* **Authentication:** Auth.js (via auth-client)
* **Deployment:** Vercel
